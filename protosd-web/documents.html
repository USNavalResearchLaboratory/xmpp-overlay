Proto-SD

ProtoSD provides a toolkit for exposing the raw functionality of JmDNS and INDI in a programming-friendly way to applications using command line integration or through Web interfaces via its REST API. The following image provides an overview of the functionality provides by ProtoSD.

ProtoSD has a Web-based application programming interface (API), a command line interface and a desktop-style Graphical User Interface (GUI) that is written using Web technologies. To use ProtoSD, all you need is an off the shelf browser or a terminal window and you can begin to advertise and discovery mDNS services. For more details, take a look at the architecture section in the User Guide
