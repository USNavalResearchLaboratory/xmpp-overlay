INDI Parameters

Please specify the parameters for INDI below. All parameters should be specified in milliseconds and are mostly single parameters. However, the client query intervals should be a set of absolute offset timers that are used by the INDI client to send queries out to the network. These should be space delimited e.g. 100 200 400 800 1600. To repeat message indefinately, use -1.
